# LabMaketUNO
Presented here is a development board design based on the Arduino UNO.<br>
This board is the author's own work; the circuit design solutions were adapted
from the technical documentation of similar boards.
The PCB layout may require adjustment to expand its functionality.<br>
The author used this board for educational purposes at their institute.<br>

Вашему вниманию предлагается проект отладочной платы на базе Arduino UNO.<br>
Данная плата является плодом труда автора, схемотехнические решения заимствованы
из технической документации других аналогичных плат.
Разводка платы, возможно, требует корректировки, для увеличения ее функционала.<br>
Данная плата использовалась в обучающих целях в институте автором.<br>
